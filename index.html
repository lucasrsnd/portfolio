Chiposto

Tecnologias utilizadas: Html, Css, Bootstrap.

Oferecer uma experiência intuitiva ao usuário, listando e classificando por preços atualizados os postos de gasolina na região solicitada, além de apresentar a classificação(estrelas, postos 24h, com coveniência) dos postos apresentados. Além disso, planejamos oferecer aos donos dos estabelecimentos uma forma de se promoverem no site.

Java Cooking

Tecnologias utilizadas: Java, Java Swing.

Gerenciamento de Restaurante Este projeto é uma aplicação Java com interface gráfica para o gerenciamento de um restaurante, facilitando o controle de clientes, pedidos, fila de espera, métricas de vendas e entregas. Funcionalidades: adicionar clientes, gerenciamento de pedidos, fila de espera, métricas de venda, delivery, formas de pagamento.

Tech Frotas

Tecnologias utilizadas: Html, Css, Bootstrap, Javascript, Java, Springboot e PostgreSql.

O projeto tem como objetivo desenvolver um sistema para otimizar a gestão de frotas dos caminhões da empresa, proporcionando um maior controle sobre todos os seus serviços, como abastecimentos, gastos, faturamento, salários, pagamentos, entre outros. Permitirá também o registro e armazenamento automatizado de todos os dados que a empresa atualmente gerencia manualmente.
